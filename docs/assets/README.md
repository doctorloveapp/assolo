# Assets Directory

This directory contains visual assets for documentation and marketing:

## Required Assets

### Logo
- `logo.png` - App icon/logo (120x120 px, transparent background)

### Screenshots
- `screenshot_main.png` - Main instrument grid screen (1080x1920 or similar)
- `screenshot_settings.png` - Settings panel expanded
- `screenshot_playing.png` - Active notes highlighted during play

### Feature Graphics (for Play Store)
- `feature_graphic.png` - 1024x500 px promotional banner
- `promo_video_thumbnail.png` - YouTube thumbnail for promo video

## Guidelines

- Use PNG format for all images
- Optimize file sizes for web display
- Screenshots should be taken on a device with minimal status bar clutter
- Consider dark mode appearance consistency

## Generating Screenshots

1. Run the app on a clean emulator or device
2. Hide the status bar notifications
3. Capture at native resolution
4. Crop if needed, but maintain aspect ratio
